
Spotify Clone
A web-based Spotify clone created using JavaScript, HTML, and CSS. This project replicates some of the core functionalities of Spotify, such as music streaming , in a simple and responsive web application.


Features
Music Playback: Play, pause, and skip tracks.
Responsive Design: Works seamlessly on desktop 
Technologies Used
HTML: Structure and content of the web pages.
CSS: Styling and layout of the web pages.
JavaScript: Functionality and interactivity of the application
